# Corporate-Bankruptcy-Prediction-An-early-warning-approach
## Objective
To identify early warning signs of bankruptcy by analyzing the financial performance of five bankrupt Indian companies.

## Tools Used
Microsoft Excel, Power BI

## Key Findings
-Identified key bankruptcy indicators such as rising debt, continuous losses, and negative cash flow.
-Analyzed financial trends across five companies using revenue, profit, debt, and cash flow data.
-Built Power BI dashboards to visualize financial distress patterns.
